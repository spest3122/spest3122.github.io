# 農業相關技術移轉的平台

## 簡介

內容涵蓋有苗栗改良場，畜產研發成果，林業試驗所，台東區農業改良場，桃園區農業改良場的技術移轉公告列表
主要目的: 提供一個平台查看技術移轉的品項與當前的授權公司以及發佈時間
